Components

This package registers the following components:

mouse

Description

This component allows you to incorporate mouse support to your game, via two point collisions for the mouse hover and click. It also generates click events.

Triggers

This component can trigger the following events:

click

This event will be triggered once the mouse is clicked

The trigger sends the following data:

{"x":"<The x coordinate of the mouse>","y":"<The y coordinate of the mouse>","button":"<Left button=0, middle button=1, right button=2>"}
