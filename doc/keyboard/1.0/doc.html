Components

This package registers the following components:

keyboard

Description

This component allows you to incorporate keyboard support to your game, sending key events that other components can listen to.

Triggers

This component can trigger the following events:

keyboardDown

This event will be triggered once a key is pressed.

The trigger sends the following data:

{"key":"<The key code>"}

keyboardUp

This event will be triggered once a key is released.

The trigger sends the following data:

{"key":"<The key code>"}
