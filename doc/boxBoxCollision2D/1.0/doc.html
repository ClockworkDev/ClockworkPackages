Collisions

This package registers the following collisions:

box - box

The collision happens when one box overlaps with the other.

The schema of box is:

{"x":"<The x coordinate of the box>","y":"<The y coordinate of the box>","w":"<The width of the box>","h":"<The height of the box>"}

The schema of box is:

{"x":"<The x coordinate of the box>","y":"<The y coordinate of the box>","w":"<The width of the box>","h":"<The height of the box>"}
