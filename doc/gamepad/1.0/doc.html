Components

This package registers the following components:

gamepad

Description

This component allows you to incorporate gamepad support to your game, translating the Gamepad API to events that other components can listen to.

Triggers

This component can trigger the following events:

gamepadAxis

This event will be triggered once per frame for each gamepad connected, sending the state of the gamepad's thumsticks.

The trigger sends the following data:

{"values":[{"x":"<X value on the first thumsbtick>","y":"<Y value on the first thumsbtick>"},{"x":"<X value on the second thumsbtick>","y":"<Y value on the second thumsbtick>"}],"player":"<The number of the gamepad connected>"}

gamepadTrigger

This event will be triggered once per frame for each gamepad connected, sending the state of the gamepad's triggers.

The trigger sends the following data:

{"leftValue":"<The value of the left trigger>","rightValue":"<The value of the right trigger>","player":"<The number of the gamepad connected>"}

gamepadDown

This event will be triggered once a gamepad button is down.

The trigger sends the following data:

{"name":"<The name of the button>","player":"<The number of the gamepad connected>"}

gamepadUp

This event will be triggered once a gamepad button is up.

The trigger sends the following data:

{"name":"<The name of the button>","player":"<The number of the gamepad connected>"}

gamepadDisconnected

This event will be triggered once a gamepad is disconnected.

The trigger sends the following data:

{"player":"<The number of the gamepad disconnected>"}
